# Beatriz Cavalcante | Treinadora de Processos

Site profissional para mentoria em gestão e produtividade.

## 🚀 Tecnologias

- Next.js 15 (App Router)
- TypeScript
- Tailwind CSS
- Framer Motion
- React Hook Form + Zod

## 📦 Instalação

```bash
# Clone o repositório
git clone https://github.com/seu-usuario/beatriz-cavalcante-site.git

# Instale as dependências
npm install

# Configure as variáveis de ambiente
cp .env.example .env.local

# Rode o projeto
npm run dev
